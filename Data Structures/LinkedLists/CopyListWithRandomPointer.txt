```python
"""

        self.val = int(x)
        self.next = next
        self.random = random

class Solution:
    def copyRandomList(self, head: 'Optional[Node]') -> 'Optional[Node]':
        oldToCopy = {None: None}

        cur = head
        while cur:
            copy = Node(cur.val)
            oldToCopy[cur] = copy
            cur = cur.next
        cur = head
        while cur:
            copy = oldToCopy[cur]
            copy.next = oldToCopy[cur.next]
            copy.random = oldToCopy[cur.random]
            cur = cur.next
        return oldToCopy[head]
```

```python
class Solution:
```

### **ğŸš€ Understanding the Copy List with Random Pointer Algorithm**

This code **efficiently copies a linked list** where each node has two pointers:

1. **`next` pointer** â†’ Points to the next node in the list.
2. **`random` pointer** â†’ Points to **any node** (or `None`) in the list.

## **ğŸ” Step-by-Step Explanation**

We use a **3-pass strategy** to copy the list **without using extra space for a hashmap**.

---

### **ğŸ“Œ Step 1: Interleave Nodes (First While Loop)**

ğŸ’¡ **Goal:** Create a **copy node** next to each original node.

### **How it Works**

1. **Iterate through the list**, creating a **copy of each node** (`l2`).
2. **Insert each copy (`l2`) right after the original node (`l1`)**.
3. **Adjust pointers** to interleave original and copied nodes.

### **Example**


```python
    l1 = l2.next       # Move to the next original node

```

### **ğŸ“Œ Step 2: Copy Random Pointers (Second While Loop)**

ğŸ’¡ **Goal:** Set the **`random` pointer** for the copied nodes.

### **How it Works**

- Each original node **already has a copy next to it**.
- If an original node has a `random` pointer:
    - **Copy its `random` pointer** to its corresponding copied node.


### **ğŸ“Œ Step 3: Separate the Copied List (Third While Loop)**

ğŸ’¡ **Goal:** Separate the copied list from the original.

### **How it Works**

1. **Extract all copied nodes** (`A' â†’ B' â†’ C' â†’ D'`).
2. **Restore the original list structure** (`A â†’ B â†’ C â†’ D`).

### **Code**

```python

    l1.next = l2.next  # Restore original list
    if l2.next is not None:
        l2.next = l2.next.next  # Connect copied nodes
    l1 = l1.next  # Move forward

```
